        - No. of Subscribers
        - No. of Likes
        - No. of Videos
    - Decimal numbers (positive, negative, zero)
        - Prices
        - Ratings

- String (everything you see on your keyboards, has more characters.

- Boolean (True: correct or False:Wrong)

- Array (list) use []

- Object use object group information and {}
